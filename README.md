## Installation
Windows
```shell
py -m venv venv
venv\Scripts\activate
py -m pip install pkg-cli
```

Linux/unix/macOS
```shell
python3 -m venv venv
source venv/bin/activate
python3 -m pip install pkg-cli
```

## Usage
Create a 'package.json' file
```shell
pkg init
```

Publish package to PyPi
```shell
pkg publish
```
